使用说明：<br />

如果是未修改版本，推荐直接重新安装，如果你不想重装继续往下看~<br />
将本包内文件解压覆盖，然后进入数据库找到setting表，清空cloud这一行的内容(删除也行)，然后清空cache表<br />
最后去系统设置刷新下看看云平台注册上没~如果有站点ID就成功了<br />
